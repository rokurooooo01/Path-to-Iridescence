# what this game is all about?

escape from your shadow, that's it.

there are 5 stages in total with each stage gets more difficult (more floors and stairs basically, i dont have the mental fortitute to add shit like precision airstrike or smth).

all you need is to run from your shadows using the classic WASD keys.

W for going up;
A for going left;
S for going down;
D for going right.

while there is indicator where the escape door is located, since this is a text based game, there isnt mini map, so this shit is pretty much try and error lmao. maybe before the due date ends i will update this game a bit more.
