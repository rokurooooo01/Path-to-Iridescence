# what this game is all about?
escape from your shadow, that's it.

there are now 10 stages in total with each stage getting progressively more difficult (more floors, bigger labyrinths, dropping floors, etc.). originally it was 5, but we cooked. (still don't have the mental fortitude to add shit like precision airstrikes or smth though).

all you need to do is survive, manage your resources, and run from your shadow using the classic WASD keys.

Controls:

W - go up / climb ladder

A - go left

S - go down / descend ladder

D - go right

F - Toggle Flashlight (keep it on to see, but it enrages the Shadow)

R - Toggle Sprint (drains stamina, but you move 2 spaces)

H - Hide in Lockers (comes with a breathing QTE so you don't die)

T - Throw stones to distract the AI

1, 2, 3 - Use items in your backpack

P - Pause the game (freezes the timer, lets you view lore)

originally this was pure trial and error with no map, but now there is a massive dynamic UI with a mini-map, FOV lighting, compass, and event logs.


# changes
as of the latest update, a massive amount of changes have been made to turn this into an actual survival horror experience. they are:

Expanded to 10 Stages with a final boss escape sequence.

Added a dynamic Shadow AI that tracks you by sound, sight, and scent.

Added Difficulty Modes (Beginner, Normal, and 1-HP Nightmare Mode).

Survival Mechanics: Sanity, Stamina, and Flashlight Battery management.

Interactables: Safe zones (Light Pillars), lockers to hide in, crumbling floors, and warp gates.

Inventory System: Health potions, energy drinks, bear traps, signal flares, and stones to throw.

Lore System: Collectable journal fragments that permanently save to your local device (spire_archive.sav) so you can read them in the Main Menu Bestiary.

Fully functional Pause Menu that properly stops the timer.

Input safety locks (so mashing random keys doesn't break the UI).

Added sound effects (QoL beep sounds).
